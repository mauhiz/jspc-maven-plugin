jspc-maven-plugin
=================

(moved from [Google Code](https://code.google.com/p/jspc-maven-plugin ))

## Usage

      <plugin>
        <groupId>net.mauhiz</groupId>
        <artifactId>jspc-maven-plugin</artifactId>
        <version>7.0.54</version>
      </plugin>

## What does it do?

Precompiles JSPs with the full maven classpath.
